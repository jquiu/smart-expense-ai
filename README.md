# 💸 Smart Expense AI
**Gestión inteligente de finanzas personales asistida por IA.**

Smart Expense AI es una aplicación web de vanguardia diseñada para centralizar, categorizar y analizar gastos personales. Utiliza inteligencia artificial para ofrecer insights financieros, ayudando a los usuarios a tomar decisiones informadas sobre su dinero.



## 🚀 Tecnologías Principales (The Stack)

Este proyecto utiliza un stack moderno (Bleeding Edge) para garantizar escalabilidad, velocidad y una experiencia de usuario fluida:

* **Framework:** [Next.js 15](https://nextjs.org/) (App Router & Turbopack).
* **Base de Datos:** [Neon](https://neon.tech/) (Serverless PostgreSQL).
* **ORM:** [Prisma](https://www.prisma.io/) (Type-safe database client).
* **Autenticación:** [Auth.js (NextAuth beta)](https://authjs.dev/) con integración de Google OAuth.
* **UI/UX:** [Tailwind CSS](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/) para una interfaz limpia y profesional.
* **IA:** Preparado para integración con modelos de lenguaje (LLMs) para el análisis de patrones de gasto.

## ✨ Características (Features)

* **Autenticación Social:** Inicio de sesión seguro con Google.
* **Control de Roles:** Sistema diferenciado para usuarios (`USER`) y administradores (`ADMIN`).
* **Persistencia Serverless:** Datos almacenados en la nube con Neon, garantizando alta disponibilidad.
* **Diseño Responsive:** Interfaz adaptable para dispositivos móviles y escritorio.
* **Gestión de Categorías:** Clasificación inteligente de gastos (Comida, Transporte, Suscripciones, Salud, etc.).
